快速上手指南

四步即可开始使用 Flomo AI Chrome 插件，简单易用

1

安装插件

从 Chrome 网上应用店安装插件，或下载离线安装包手动安装到浏览器中

2

配置服务

在插件设置页面配置 OpenRouter API Key 和 Flomo API URL，支持多种AI服务

3

选择文本

在任意网页中选中想要总结的文本内容，右键菜单或点击插件图标

4

一键保存

AI自动分析并生成摘要，确认后一键保存到您的 Flomo 笔记库中
